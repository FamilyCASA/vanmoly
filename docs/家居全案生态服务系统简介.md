# 家居全案生态服务系统简介

## 一、系统定位

**VANMOLY 家居全案生态服务系统 V3.0** 是一款专为家居装饰行业深度定制的一站式数字化管理平台。系统面向家居全案服务企业（整装公司、设计工作室、高端定制机构），打通"获客→设计→报价→合同→施工→交付"的全业务流程，实现前后台一体化、管理运营智能化。

系统采用 **Flask + Vue3 + Element Plus** 技术架构，支持多角色协同（客户、设计师、项目经理、施工队、管理员），覆盖从案例展示、智能选品、在线预约、报价合同管理到员工考核的完整生态。

## 二、核心功能模块

### 2.1 前台门户（客户侧）

| 模块 | 功能说明 |
|------|---------|
| **案例展示** | 沉浸式案例浏览，支持按楼盘、面积、风格、预算多维筛选；Hero 大图展示，色卡直观呈现空间氛围 |
| **智能选品** | 客户自主选品中心，支持按品类、风格、材质筛选；选品单实时同步至后台 |
| **楼盘数据库** | 覆盖本地楼盘信息，支持楼盘调查录入，精准匹配客户需求 |
| **在线预约** | 客户填写需求表单，自动生成线索，分配至对应设计师 |
| **关于我们** | 企业介绍、服务流程展示（三层同心圆设计），建立品牌信任 |

### 2.2 后台管理（员工侧）

| 模块 | 功能说明 |
|------|---------|
| **仪表盘** | 经营数据可视化，线索转化漏斗、项目进度、业绩统计 |
| **案例管理** | 案例 CRUD、多图上传、AI 智能标签、案例上下架 |
| **客户管理** | 客户档案、跟进记录、需求标签、客户生命周期管理 |
| **线索管理** | 线索分配、跟进提醒、转化率分析、ROI 统计 |
| **报价管理** | 快速报价、报价模板、材料清单自动生成、利润核算 |
| **合同管理** | 合同起草、电子签章、付款进度跟踪、合同归档 |
| **物料管理** | SKU 管理（728+ 条物料数据）、库存预警、供应商管理 |
| **员工管理** | 员工档案、权限管理、绩效看板、提成计算 |
| **工作流引擎** | 自定义审批流程、节点提醒、流程模板管理 |
| **知识库** | 企业知识沉淀、案例复盘、标准 SOP 文档管理 |

## 三、商业价值

### 3.1 降本增效

- **流程自动化**：从线索到交付全流程数字化，减少人工传递信息误差，平均缩短项目周期 **20%**
- **选品标准化**：客户在线选品，减少线下沟通成本，设计师出图效率提升 **35%**
- **报价智能化**：系统自动套用物料库价格，一键生成报价单，报价时间从 **2 小时 → 15 分钟**

### 3.2 提升转化

- **案例展示专业化**：沉浸式案例页，客户停留时长提升 **3 倍**，线索转化率提升 **25%**
- **快速响应**：在线预约 + 自动分配，线索响应时间从 **24 小时 → 2 小时**
- **品牌信任建立**：关于页面服务流程可视化，降低客户决策门槛

### 3.3 数据驱动决策

- **经营看板**：实时掌握线索来源、转化率、项目毛利等核心指标
- **员工绩效**：自动统计设计师成单率、客单价，为薪酬激励提供数据支撑
- **物料成本分析**：精准核算项目成本，及时发现物料浪费和价格异常

## 四、行业深度定制优势

### 4.1 懂家居行业的语言

本系统不是通用 CRM 的"改头换面"，而是从零按照家居全案服务的工作习惯设计：

- **案例 = 最核心资产**：系统以"案例"为中心组织数据，而非传统的"客户"中心。因为在家居行业，案例就是最好的销售工具。
- **选品单 = 设计沟通载体**：客户选品单不仅是购物清单，更是设计师理解客户审美的桥梁。系统支持选品单实时协作，设计师可一键将选品单转为报价单。
- **楼盘调查 = 前置服务**：系统内置楼盘数据库和调查模板，设计师在首次沟通前即可掌握户型优缺点，建立专业形象。

### 4.2 覆盖"全案服务"独特场景

| 独特场景 | 通用系统 | 本系统 |
|---------|---------|--------|
| 案例多维检索（楼盘/面积/风格/预算） | ❌ 不支持 | ✅ 原生支持 |
| 选品中心（客户自主选品） | ❌ 不支持 | ✅ 原生支持 |
| 报价自动关联物料库 | ❌ 需定制开发 | ✅ 开箱即用 |
| 合同进度与客户门户联动 | ❌ 不支持 | ✅ 原生支持 |
| 工作流适配装修流程（量房→设计→报价→合同→施工） | ❌ 不支持 | ✅ 流程模板内置 |

### 4.3 技术架构适配行业特性

- **图片密集型优化**：案例展示涉及大量高清图，系统采用懒加载 + WebP 自适应，页面加载速度提升 **60%**
- **移动办公支持**：设计师现场量房、客户家拜访，可通过移动端快速录入信息，不掉链子
- **数据私有化部署**：支持本地服务器部署，案例数据库、客户信息不泄露，符合家装行业对客户隐私的高要求

### 4.4 可持续演进

- **模块化设计**：功能模块解耦，企业可根据发展阶段按需启用（如初创期只用案例+线索，发展期加入报价+合同）
- **开放 API**：支持与酷家乐、三维家等设计软件对接，设计图可一键同步至系统案例库
- **AI 就绪**：系统已预留 AI 接口，未来可接入智能设计推荐、自动生成设计方案描述等功能

## 五、典型使用场景

**场景一：新客户线索跟进**
1. 客户在门户提交预约 → 系统自动创建线索，分配至对应设计师
2. 设计师收到企微/短信提醒 → 登录系统查看客户需求和楼盘信息
3. 设计师上门量房 → 移动端录入楼盘调查数据
4. 回到公司 → 从楼盘库调取同类案例，快速匹配客户喜好
5. 系统生成初步方案和报价 → 发送给客户确认

**场景二：老客户转介绍**
1. 老客户登录客户门户 → 查看自己家的完工案例
2. 一键分享至朋友圈 → 带来新的高质量线索
3. 新线索标记"转介绍" → 系统自动计算老客户积分/优惠

**场景三：项目进度透明化**
1. 项目经理每日更新施工进度 → 上传现场照片
2. 客户门户实时查看 → 减少沟通成本，提升客户满意度
3. 关键节点（水电验收、泥木验收）系统自动提醒客户和项目经理
