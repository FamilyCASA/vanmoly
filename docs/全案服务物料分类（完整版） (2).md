# 全案服务物料分类（完整版）

# 一、核心升级：新建物料分类优化

新建物料时，首先选择「物料分类」下拉菜单，明确一级分类与对应二级分类，分类体系规范如下，下拉菜单按以下层级展示，便于快速选择、规范管理，覆盖全案服务全流程物料及服务类型，无遗漏、无交叉：

## 1\. 一级分类：全案服务

二级分类（下拉可选，单一选择）：

- 全案规划提案服务（含方案构思、需求对接、整体统筹）

- 效果图设计服务（含空间效果图、软装搭配效果图、细节渲染）

- 施工图设计服务（含平面布局图、立面图、节点大样图、水电施工图）

- 家具深化设计服务（含固装/成品家具细化设计、尺寸核对、工艺说明）

- 现场测绘服务（含原始户型测绘、现场复核、尺寸确认）

- 全案预算服务（含分项预算、汇总预算、变更预算、结算核对）

- 全案落地跟进服务（新增，含施工对接、物料验收、进度管控）

## 2\. 一级分类：硬装主材

二级分类（下拉可选，单一选择，包含以下类别及延伸项）：

- 地面主材（地砖、地板、仿古砖、通体砖、实木地板、复合地板、强化地板）

- 墙面/顶面主材（涂料、乳胶漆、艺术漆、硅藻泥、石膏板材、木工板材、龙骨）

- 水电主材（电缆、电线、水管、布线管、管件、线盒、漏电保护器）

- 门窗主材（封窗、断桥铝窗、系统窗、安全门、入户门、过门石）

- 电器主材（照明电器、家用电器、暖通电器、新风设备、净水设备）

- 卫浴主材（卫浴设备、马桶、花洒、浴室柜、浴缸、集成吊顶、浴霸）

- 其他相关硬装主材（可根据实际需求补充，如石材、马赛克、踢脚线）

## 3\. 一级分类：硬装施工服务

二级分类（下拉可选，单一选择，包含以下类别及延伸项）：

- 结构施工服务（现浇楼梯、钢构搭建、混凝土浇筑、楼板加固）

- 水电施工服务（防水工程、地面找平、水电改造、管线预埋、打压测试）

- 造型施工服务（吊顶施工、木工造型、背景墙制作、柜体基础搭建）

- 安装施工服务（设备安装、门窗安装、卫浴安装、吊顶安装）

- 收尾施工服务（墙面涂装、成品保护、开荒保洁、细节修补）

- 其他相关硬装施工服务（可根据实际需求补充，如石材安装、瓷砖美缝）
