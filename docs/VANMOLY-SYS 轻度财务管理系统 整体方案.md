# VANMOLY\-SYS 轻度财务管理系统 整体方案

## 一、系统定位

嵌入现有梵木里 VANMOLY\-SYS 主系统，**轻量化不重构**，深度关联已有核心模块：楼盘 / 客户 / 案例 / 合同 / 员工 / 积分 / 知识库摘星奖惩规则；实现**营收回款管控、门店收支流水、银行流水对账、智能周期报表、员工考勤绩效薪资一体化**。

## 二、关联已有模块打通逻辑

### 1\. 楼盘管理关联

- 按**楼盘维度**统计：该楼盘所有客户合同金额、已回款、待回款、未签约潜在营收

- 楼盘成本分摊：门店拓客成本、异业合作成本、楼盘物料投放成本归集

### 2\. 客户管理关联

- 客户档案绑定财务台账：意向金、定金、首付款、中期款、尾款、增项款全记录

- 自动标记**客户回款状态**：未回款、部分回款、已结清、逾期待催款

- 客户历史消费、复购潜力、欠款记录一键查询

### 3\. 案例管理关联

- 精品案例 / 套餐案例绑定**标准报价体系**

- 案例成交后自动同步合同金额至财务模块，作为营收数据源

- 按案例类型（整装 / 定制 / 局改）统计营收占比、利润率

### 4\. 合同管理关联（核心打通）

- 合同一键同步财务：合同总金额、付款节点、约定回款时间、增项金额、优惠金额

- **合同执行进度 \+ 资金回款进度双联动**：

    1. 合同立项 → 财务生成应收台账

    2. 每阶段施工节点完成 → 自动触发对应款项待收提醒

    3. 回款到账 → 回填合同状态、更新执行进度、关闭待收金额

- 逾期合同自动预警：超约定回款日未到账，弹窗 \+ 消息提醒

### 5\. 员工管理关联

- 员工档案绑定薪酬体系：底薪、提成比例、岗位津贴、扣款规则

- 绑定考勤、绩效、接单业绩、案例成交贡献

### 6\. 积分管理 \&amp; 知识库摘星 / 奖励规则关联

- 知识库学习摘星、答题积分、门店贡献积分 **自动折算绩效加分 / 现金奖励**

- 违规扣分、服务差评扣分 自动关联薪资扣款

- 积分奖惩直接同步到员工薪资核算台账，无需手动二次录入

## 三、核心功能模块设计

### （一）客户合同资金回款管理

1. 应收台账：自动抓取所有生效合同，拆分**定金 / 首付 / 中期 / 尾款 / 增项**多期应收

2. 回款登记：手动 \+ 批量录入到账金额、付款方式、到账时间、凭证上传

3. 逾期催款：自动统计逾期天数、逾期金额，支持一键导出催款名单

4. 回款统计：按门店、业务员、楼盘、时间维度统计回款率、未回款占比

### （二）门店支出流水管理

- 日常支出分类：物料采购、房租水电、营销推广、异业合作、员工福利、办公耗材、工程杂费

- 每日流水登记：日期、支出类别、用途、金额、经办人、凭证照片、关联楼盘 / 项目

- 支出审批流：登记→店长审核→财务归档，可设置权限

### （三）银行流水对账管理

1. 支持手动录入 \+ Excel 批量导入银行对公 / 私户流水

2. 自动匹配：银行到账流水 ↔ 客户合同回款，一键对账核销

3. 未匹配流水单独列表，人工认领归属客户 / 项目 / 支出

4. 留存银行流水台账，可跨年查询、导出凭证

### （四）智能自动周期报表（合规化）

系统**自动定时生成**，支持预览、导出 Excel/PDF、打印存档：

- 日报：当日收支、当日回款、当日新增合同应收、当日员工考勤异常

- 周报：周营收、周支出、回款完成率、楼盘成交排行、员工业绩排行

- 月报：月度财务盈亏、各门店营收对比、合同回款履约率、成本结构分析

- 季报 / 年报：年度营收趋势、利润率、各业务板块占比、人力成本占比、楼盘投放 ROI

报表内置固定合规模板，无需手动做表，数据全部取自系统已录入台账。

### （五）员工考勤 \+ 绩效 \+ 薪资一体化管理

1. 考勤管理：上下班打卡、请假、旷工、迟到早退自动统计

2. 绩效核算：

    - 成交合同提成、案例推荐提成、楼盘拓客提成

    - 关联知识库**摘星奖励、积分奖励、扣分扣款**自动带入

3. 薪资自动核算：
底薪 \+ 绩效提成 \+ 津贴 \+ 积分奖励 \- 请假扣款 / 违规扣款 = 实发工资

4. 薪资台账：每月薪资明细存档，可给员工自助查看、导出工资条

## 四、系统权限设计

1. 超级管理员：全模块查看、修改、报表导出、权限配置

2. 财务专员：流水登记、回款录入、对账、报表查看，不可改合同与员工资料

3. 门店店长：查看本店收支、本店员工考勤绩效、本店合同回款，无全局财务数据

4. 普通员工：仅查看个人考勤、个人积分、个人薪资明细
