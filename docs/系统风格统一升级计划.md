## 一、目标

将所有页面统一到 **「暗黑深空」** 风格：
1. 统一暗色主题（暗黑深空配色）
2. 所有页面 Web 端全屏显示（含弹窗）
3. 宽度自适应不同终端设备
4. 字体颜色与背景色对比度充足，不混淆
5. 每完成一步在下方 Checklist 标记 `完成`

- 有些页面全屏，有些约束宽度
- 弹窗样式不统一
- 字体颜色可能和背景接近（需逐一检查）

## 三、技术方案

### 3.1 暗黑深空配色方案

```css
:root.dark {
  /* 背景色系 - 深空 */
  --bg-base: #0a0a1a;           /* 最底层背景：极深蓝黑 */
  --bg-surface: #1a1a2e;        /* 表面/卡片背景 */
  --bg-surface-hover: #22223a;   /* 表面悬浮 */
  --bg-elevated: #16213e;       /* 悬浮层/弹窗背景 */
  
  /* 主色调 */
  --primary: #409EFF;            /* Element Plus 蓝（保留） */
  --primary-dark: #2a6cb0;       /* 主色暗色 */
  --accent: #6C63FF;            /* 深空紫蓝强调色 */
  
  /* 文字色系 - 确保与背景对比度 */
  --text-primary: #E8E8E8;      /* 主文字：浅灰白，对比度充足 */
  --text-title: #FFFFFF;         /* 标题：纯白 */
  --text-secondary: #A0A0B8;     /* 辅助文字：浅灰蓝 */
  --text-placeholder: #606080;   /* 占位文字 */
  
  /* 边框/分割色 */
  --border: #2a2a3e;
  --divider: #1a1a2e;
  
  /* 阴影 */
  --shadow: 0 2px 12px rgba(0, 0, 0, 0.6);
}
```

### 3.2 全屏布局方案

- 所有页面根元素：`min-height: 100vh; width: 100%;`
- 内容区域：`max-width: 1400px; margin: 0 auto; padding: 20px;`（大屏约束宽度，小屏全屏）
- 弹窗：设置 `fullscreen` 或 `width: 90vw; max-width: 1200px;`

### 3.3 Element Plus 暗黑模式

使用 Element Plus 内置暗黑模式：
```js
// main.js
import 'element-plus/theme-chalk/dark/css-vars.css'
```

```css
/* 全局 */
html.dark {
  color-scheme: dark;
}
```
