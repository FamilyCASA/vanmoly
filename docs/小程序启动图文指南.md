## 第一步：启动后端服务

1. 按 `Win + R`，输入 `cmd`，回车打开命令提示符
2. 输入以下命令：
```
cd D:\desktop\DESIGNARY-SYS-V3.0\backend
python run.py
```
3. 看到 `Running on http://127.0.0.1:5000` 表示成功
4. **不要关闭这个窗口**

---

## 第二步：打开微信开发者工具

双击桌面上的 **「微信开发者工具」** 快捷方式

---

## 第三步：导入小程序项目

### 3.1 点击「+」号
在打开的界面中，找到并点击 **「+」** 按钮（通常在中间或右侧）

### 3.2 填写项目信息
在弹出的窗口中填写：

| 字段 | 填写内容 |
|------|---------|
| 项目名称 | D&B 帝标|设记家小程序 |
| 目录 | 点击选择文件夹 → D:\desktop\DESIGNARY-SYS-V3.0\miniapp |
| AppID | 选择「测试号」或填写 `wx1234567890abcdef` |
| 后端服务 | 选择「不使用云服务」 |

### 3.3 点击「确定」

---

## 第四步：关闭 HTTPS 校验（重要！）

因为后端是 HTTP 不是 HTTPS，必须关闭校验：

1. 点击右上角的 **「设置」** 图标（⚙️ 齿轮形状）
2. 在左侧菜单选择 **「项目设置」**
3. 找到 **「本地设置」** 标签页
4. ✅ 勾选 **「不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书」**

---

## 常见问题

### Q: 提示「服务端口已关闭」
A: 在微信开发者工具中：
1. 点击「设置」→「安全设置」
2. 开启「服务端口」

### Q: 提示「无法连接到后端」
A: 确保：
1. 后端服务已启动（cmd窗口显示Running on...）
2. 已勾选「不校验合法域名」
3. 后端地址是 http://127.0.0.1:5000

### Q: 模拟器显示空白
A: 检查：
1. 项目目录是否正确（D:\desktop\DESIGNARY-SYS-V3.0\miniapp）
2. 查看控制台是否有报错（右侧或下方）
