# 测试线索数据使用说明

## 生成测试数据

```bash
cd D:\desktop\DESIGNARY-SYS-V3.0\backend
python seed_test_leads.py
```

```bash
cd D:\desktop\DESIGNARY-SYS-V3.0\backend
python seed_test_leads.py --clear
```
