## 技术要点

### WXS 工具模块已就位
`utils/util.wxs` 提供了 `toFixed(value, digits)`、`includes(arr, item)`、`isSelected(selectedItems, id, variant)`，用于 WXML 中替代 JS 方法调用。

### 页面调用 API 方式
- `app.request({ url: '/frontend/hero-slides', ... })` → 实际请求 `http://localhost:8080/api/v3/frontend/hero-slides`
- `app.resolveImageUrl(path)` → 直接拼接完整 URL

### 组件替换对照
| Vue 组件 | 小程序替代 |
|---|---|
| `el-button` / `el-form` | 原生 `view` + `input` + `picker` |
| `el-dialog` | 自定义 `view.modal` |
| `v-for` | `wx:for` |
| `v-if` / `v-show` | `wx:if` |
| Vue computed | JS `data` + `setData` |
| `router.push()` | `wx.navigateTo` / `wx.switchTab` |
